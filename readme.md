# DevCAmper API
> Backend API's for DevCamper Applications

## Usage
Rename "config/config.env.env" to "config/config.env" and update values and settings as your wish

## Install Dependancies
npm install
## Run App
# Run in dev mode
npm run dev
# Run in Production
npm start
